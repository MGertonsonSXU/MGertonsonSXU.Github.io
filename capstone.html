Capstone Project

Project Title: TechX

The project me and my group had worked on was TechX. TechX was designed as a website revolving around technology. TechX uses Node.js, Express, MySQL, and a frontend built with HTML, CSS, and JavaScript, supported by REST APIs and session‑based authentication. My role was the team leader, where I was to check on my team and make sure we made deadlines. Additionally, I was mainly responsible for two features overall: The reviews system, where users can leave reviews on products(and add image support, filtering, and an average review), and the shopping cart system(which encompasses the cart itself, adding items to the cart, completing the order, storing the order, and havuing a purchase history.

Working on TechX taught me how a real full-stack app can be created. I got experience working with Multer for file uploads, making API routes, and dealing with SQL date queries. Additionally, I got more experience working with async functions and handling data flow between the front and back ends. Working on the reviews and shopping cart features would push me even further, in particular I learned to structure multi-table queries, calculate averages for ratings, manage the cart state, and debugging issues that come along the way. Overall, it was a great crash course in building something that actually feels like a real product.
